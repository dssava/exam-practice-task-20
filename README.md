# Пекарня: відстеження виробництва пончиків

## Опис
Додаток відображає процес виготовлення пончиків різних сортів. Оператор обирає сорти, задає потрібну кількість і запускає виробництво. Кожні 10 секунд виготовляється партія по 50 пончиків кожного обраного сорту, а таблиця оновлюється автоматично.

## Функціональність
- Вибір сортів пончиків і планових обсягів виробництва
- Автоматичне оновлення статистики кожні 10 секунд
- Показано план, факт та залишок по кожному сорту

## Вимоги
- .NET 8 SDK
- macOS, Windows або Linux (UI на Avalonia)

## Запуск
У терміналі з кореня проєкту:

```bash
cd csharp-bakery
dotnet run
```

Або з будь-якого каталогу:

```bash
dotnet run --project csharp-bakery/BakeryTracker.csproj
```

## Структура проєкту
- `Domain.cs` — сутність пончика та перелік сортів
- `Production.cs` — логіка плану і партійного виробництва
- `UiModels.cs` — моделі для таблиці та локалізація назв сортів
- `MainWindow.cs` — інтерфейс та керування процесом
- `Program.cs`, `App.cs` — точка входу і ініціалізація Avalonia

## Примітки
- Розмір партії: 50 штук на сорт
- Інтервал виготовлення: 10 секунд
